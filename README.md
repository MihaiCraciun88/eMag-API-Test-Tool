# eMag-API-Test-Tool
 Tool for testing eMag API (because they don't provide any)

## Dependencies
* php
* [composer](https://getcomposer.org/)

## Install
```
composer update
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
